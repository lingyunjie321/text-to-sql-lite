# docs 文档导读

本目录存放项目说明文档。按以下顺序和定位阅读。

## 推荐阅读顺序

### 1. 快速了解（先读这几个）

| 文档 | 定位 | 适合读者 |
| --- | --- | --- |
| [整体架构](整体架构.md) | 系统分层、架构全景图、工作流核心设计、状态与 Trace、数据边界 | 所有读者，第一次了解项目必读 |
| [文本转 SQL 工作流](文本转SQL工作流.md) | `workflow.yaml` 配置、节点职责表、流转图、成功/修复/终止路径 | 想了解主链路如何流转的读者 |
| [项目结构与模块职责](项目结构与模块职责.md) | 完整目录树、核心模块职责表、配置与数据文件、测试结构 | 想找代码在哪的读者 |
